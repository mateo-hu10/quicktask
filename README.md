# QuickTask

QuickTask es una aplicación web minimalista para convertir tareas repetitivas en hábitos organizados.

## Características

- Crear tareas con frecuencia diaria, semanal o mensual
- Marcar tareas como completadas
- Eliminar tareas
- Contador de tareas pendientes
- Guardado automático usando localStorage

## Tecnologías

- HTML
- CSS
- JavaScript
- localStorage

## Cómo usar

1. Abre index.html en tu navegador
2. Escribe una tarea
3. Selecciona frecuencia
4. Presiona "Agregar"
5. Marca como completada cuando la hagas
